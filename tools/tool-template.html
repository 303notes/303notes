Convert Amazon FBM Orders to DHL CSV

A small tool that converts Amazon FBM order exports into a DHL-compatible CSV file.

Problem

Amazon FBM order exports cannot be imported directly into DHL, which leads to manual data entry and shipping errors.

Solution

This tool converts Amazon FBM order exports into a DHL-compatible CSV file. No manual editing required.

Status

Works for me. Minimal and rough. No guarantees.

Who this is for

Amazon FBM sellers shipping orders via DHL.
