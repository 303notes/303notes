Convert Amazon FBM Orders to DHL CSV

A small tool that converts Amazon FBM order exports into a DHL-compatible CSV file.

Problem

I sell products on Amazon in Germany and ship most of my items via FBA (Fulfillment by Amazon), but some ASINs I ship via FBM (Fulfillment by Merchant). I have a DHL Business account, but there is no easy or affordable integration between Amazon Seller Central and DHL. As a result, I had to manually create up to 10 shipments per day, repeatedly copying the recipient‚Äôs name, company, street, house number, postal code, and selecting the correct parcel template. This was a boring, repetitive task that took me around 30 minutes every day.

Solution

I vibe-coded a simple tool that converts Amazon delivery slips into a DHL-compatible CSV import.

My workflow now looks like this:

  1. Download the delivery slip as an HTML file
  2. Upload it to the FBM2DHL tool
  3. Generate a CSV for DHL "üì¶ DHL shipment data import"
  4. Upload the CSV to DHL and generate all shipping labels
  5. Manually add the tracking numbers back into Seller Central

This saves me 10‚Äì30 minutes every day, and for the first time I don't dread creating shipping labels anymore.

Who is this for

Amazon sellers who mostly ship one ASIN per order via FBM.

Status & Limitations

  • If you usually ship one ASIN per order, this tool can save you a significant amount of time.
  • You can define unique carton dimensions and weights and assign one or more ASINs to each carton size.
  • When an ASIN is detected in the delivery slip, it is automatically mapped to the corresponding package configuration.

For orders containing multiple different ASINs, a more advanced mapping logic would be required.
